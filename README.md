# Portfolio Desktop Experience

A modern portfolio built with React and Vite that presents skills, projects, articles, resume, and contact details through a polished desktop-inspired UI.

## What this portfolio uses

- React 19 for building interactive UI components
- Vite for fast development and optimized production builds
- Tailwind CSS for utility-first styling and responsive layouts
- GSAP for smooth entrance, hover, and drag animations
- Zustand for lightweight window state management
- React Tooltip for hover interactions on the dock
- React PDF for rendering resume content
- Lucide React icons for crisp UI icons
- Day.js for lightweight date formatting
- Immer for immutable state updates
- clsx for conditional class names

## Key features

- macOS-style dock with animated icon magnification on hover
- draggable windows with open / close behavior and layered z-index control
- custom desktop app sections including:
  - Portfolio browser
  - Articles / blog post viewer
  - Photo gallery
  - Resume viewer
  - Terminal-style skills window
  - Contact and archive pages
- responsive layout with clean modern styling
- subtle animations and motion throughout the interface

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

## Notes

This project is a portfolio showcase with a strong emphasis on interaction design, animated windows, and a desktop-inspired navigation experience. It is ideal for demonstrating frontend skills, UI animation, and polished presentation.
